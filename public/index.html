How to Use

Click anywhere Start typing to create an entry
Single click entry Edit the entry
Right-click entry Edit existing entry
Enter Save the current entry
Shift + Enter New line within an entry
Clear text + Enter Delete the entry
Double click entry Navigate into entry (open subpage)
Shift + Drag Draw a box to select multiple entries
⌘/Ctrl + Click entry Add or remove entry from selection (or open link if entry has URL)
Click and drag entry Move an entry (or all selected entries)
Delete / Backspace Delete selected entries
⌘/Ctrl + Z Undo last action (move, edit, delete)
Drag canvas Pan around the canvas
Scroll Zoom in and out
⌘/Ctrl + Shift + 1 Navigate back to home page
Drag and drop image Drop an image on the canvas to place it
Paste URL in entry Auto-generates a link preview card
Media toggle (toolbar) Search for movies or songs to create media cards
LaTeX toggle (toolbar) Type plain English, converts to rendered math
